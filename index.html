如何用hugo搭建个人博客

Step1 下载hugo.exe 重启电脑后，在VSCode新建终端输入 $ hugo version 若出现版本号，则成功； 进入hugo官网，点击Quick Start 按照Step2~Step7抄代码 Step2 生成一系列目录 在一个安全位置搭建博客 $ hugo new site github用户名.github.io.creator 添加一个主题 $ git init $ git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke $ echo 'theme = "ananke"' >> config.toml Step4 创建一篇博文 $ hugo new posts/博文名称.md 当draft为true时，博文是草稿 Step5 本地预览 $ hugo server -D Step6 在VSCode中点击config.toml文件，将languageCode改为”zh-Hans” Step7 创建public目录 在不关闭第一个终端的前提下，新建一个终端 $ hugo 上传至Github Step8 新建一个.gitignore文件，将/public/写入.gitignore $ cd public $ git init $ git add .
